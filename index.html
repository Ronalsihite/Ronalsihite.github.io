Kenalan dengan Ronal

Halo, saya Ronal Maruhal Sihite, mahasiswa aktif di Program Studi Perpustakaan dan Sains Informasi, Universitas Sumatera Utara. Saya memiliki ketertarikan besar terhadap bagaimana teknologi informasi dapat dimanfaatkan dalam pengelolaan pengetahuan dan pengembangan Perpustakaan digital. Melalui situs web ini, saya ingin membagikan wawasann literasi informasi, pemanfaatan sumber Referensi ilmiah secara daring serta pentingnya akses terbuka terhadap pengetahuan. Di Sisi lain, saya juga sedang mendalami pengembangan Web menggunakan HTML, CSS JavaScript, dan Bootstrap sebagai Upaya menggabungkan kompetensi Kepdengan keterampilan digital. Harapan saya, kombinasi ini dapat Memperbaiki terciptanya Perpustakaan yang lebih modern dan dapat diakses kapan saja oleh siapa saja.
